Edge Cases:
add_num(double x, double y):

Overflow/underflow when x and y are extremely large/small.

NaN (Not-a-Number).

subtract_num(double x, double y):

Overflow/underflow.

NaN.

multiply_num(double x, double y):

Overflow/underflow.

NaN.

divide_num(double x, double y):

Division by zero.

Overflow/underflow.

NaN.

percent_num(double x, double y):

Division by zero.

NaN.

power(double base, double exponent):

Negative base with a non-integer exponent results in complex numbers.

Zero base with a negative exponent (division by zero).

NaN.

root(double base, double rootval):

Negative base with an even root results in complex numbers.

Zero rootval (division by zero).

NaN.

exp(double x):

Overflow for large x values.

Underflow for small x values.

NaN.

logarithm_e(double x):

x <= 0 (logarithm of non-positive numbers).

NaN.

logarithm_10(double x):

x <= 0.

NaN.

logarithm_2(double x):

x <= 0.

NaN.

logarithm_user_def(double x, double log_base):

x <= 0.

log_base <= 0 or log_base == 1 (logarithm base of 1 is undefined).

NaN.

antilogarithm(double x):

Overflow for large x values.

NaN.

factorial(int n):

Negative n (factorial of negative numbers is undefined).

Overflow for large n values.

NaN.

cos(double x):

NaN.

sin(double x):

NaN.

tan(double x):

x = (2k+1) * (π/2) where k is an integer (undefined as tan(x) approaches infinity).

NaN.
